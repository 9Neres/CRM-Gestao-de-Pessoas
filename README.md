# Gerenciador de Equipes - Iriy

Plataforma de análise de desempenho e gestão de pessoas focada em equipes de desenvolvimento, inspirada em fluxos de trabalho de ferramentas como Jira.

![Dashboard de CRM](https://images.pexels.com/photos/1181467/pexels-photo-1181467.jpeg)

## Tecnologias

- Next.js e React para interface web.
- TypeScript para tipagem estática e melhor manutenção do código.
- JSON como fonte de dados simulando integrações com APIs.
- Gemini CLI para geração e auxílio na criação de componentes e fluxos. 
- Deploy contínuo na Vercel para hosting e CI/CD.

## Funcionalidades

- Dashboard com visão geral de tickets (abertos, em andamento, aguardando e finalizados).
- Acompanhamento de SLAs, alertas de risco e notificações em tempo real.
- Métricas de desempenho da equipe (code reviews, PRs, cobertura de testes, resolução de SLA).
- Gestão de membros da equipe e visão por cliente/empresa.

## Objetivo

Ajudar squads de desenvolvimento a terem mais controle sobre o que está sendo feito, priorizando demandas e monitorando a saúde do time de forma visual e intuitiva.
